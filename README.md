## :envelope: Bandeja App

![Kotlin](https://img.shields.io/badge/Kotlin-2.0+-blue.svg?logo=kotlin)
![Jetpack Compose](https://img.shields.io/badge/Jetpack%20Compose-Material%203-4285F4.svg?logo=android)
![Architecture](https://img.shields.io/badge/Architecture-MVVM%20%2B%20UDF-success.svg)
![Adaptive](https://img.shields.io/badge/UI-Adaptive%20Layouts-orange.svg)

Una aplicación nativa de demostración para Android que implementa una **Bandeja de Entrada de correos**. Este proyecto está diseñado para demostrar el dominio en la creación de interfaces de usuario **adaptables** utilizando Jetpack Compose, reaccionando dinámicamente a diferentes tamaños de pantalla (Teléfonos, Tablets y Plegables).
